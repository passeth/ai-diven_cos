Journalists Manager

Manage journalist profiles, avatars, and information. Changes are saved to local files.

How to Update Journalist Profiles

  1. Change Avatar Image: Click the camera icon on the profile picture to upload a new image (PNG, JPG, or WebP recommended, 400x400px)
  2. Edit Info: Modify name, role, or bio directly in the form fields
  3. Save Changes: Click "Save" to update. This will modify build.js
  4. Rebuild Site: After saving, run npm run build to regenerate pages

File Locations

Avatar Images: content/_assets/personas/[slug].svg (or .png, .jpg, .webp)

Journalist Data: site/src/build.js (PERSONAS object)
